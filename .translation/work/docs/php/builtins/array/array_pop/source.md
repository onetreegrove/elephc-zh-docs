## array_pop()

```php
function array_pop(array $array): mixed
```

Lowers `array_pop()` for indexed arrays by mutating length and boxing `T|null` as Mixed.

**Parameters**:
- `$array` (`array`), passed by reference

**Returns**: `mixed`
