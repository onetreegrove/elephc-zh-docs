## array_search()

```php
function array_search(mixed $needle, array $haystack, bool $strict): mixed
```

针对具有类整数负载的索引数组，降级 `array_search()`。

**参数**：
- `$needle` (`mixed`)
- `$haystack` (`array`)
- `$strict` (`bool`)，可选

**返回值**：`mixed`
