## arsort()

```php
function arsort(array $array, int $flags): bool
```

Lowers `arsort()` for indexed integer arrays through the descending value-sort wrapper.

**Parameters**:
- `$array` (`array`), passed by reference
- `$flags` (`int`)

**Returns**: `bool`
