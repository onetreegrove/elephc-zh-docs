## rsort()

```php
function rsort(array $array, int $flags): bool
```

Lowers `rsort()` for indexed integer arrays by mutating the source array in place.

**Parameters**:
- `$array` (`array`), passed by reference
- `$flags` (`int`)

**Returns**: `bool`
