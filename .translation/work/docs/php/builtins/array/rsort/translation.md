## rsort()

```php
function rsort(array $array, int $flags): bool
```

通过原地修改源数组，为索引整数数组降级处理 `rsort()`。

**参数**：
- `$array` (`array`)，引用传递
- `$flags` (`int`)

**返回值**：`bool`
