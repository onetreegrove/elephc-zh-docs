## get_parent_class()

```php
function get_parent_class(mixed $object_or_class): string
```

`get_parent_class()` 是 Elephc 支持的类内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$object_or_class` (`mixed`)，可选

**返回值**：`string`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
