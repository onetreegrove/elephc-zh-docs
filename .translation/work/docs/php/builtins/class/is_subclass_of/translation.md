## is_subclass_of()

```php
function is_subclass_of(mixed $object_or_class, string $class, bool $allow_string): bool
```

`is_subclass_of()` 是 Elephc 支持的类内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$object_or_class` (`mixed`)
- `$class` (`string`)
- `$allow_string` (`bool`)，可选

**返回值**：`bool`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
