## trait_exists()

```php
function trait_exists(string $trait, bool $autoload): bool
```

`trait_exists()` 是 Elephc 支持的类内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$trait` (`string`)
- `$autoload` (`bool`)，可选

**返回值**：`bool`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
