## min()

```php
function min(mixed $value, ...$values): float
```

针对具体的类整型和浮点型操作数下放数值型 `min()` 和 `max()`。

**参数**：
- `$value` (`mixed`)
- `...$values` — 可变参数：将多余的参数收集到 `$values` 中。

**返回值**：`float`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 了解使用模式。_
