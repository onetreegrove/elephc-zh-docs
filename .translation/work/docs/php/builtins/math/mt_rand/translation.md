## mt_rand()

```php
function mt_rand(int $min, int $max): int
```

`mt_rand()` 是 Elephc 支持的数学内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$min` (`int`)
- `$max` (`int`)

**返回值**：`int`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 了解使用模式。_
