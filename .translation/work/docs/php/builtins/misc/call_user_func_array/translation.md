## call_user_func_array()

```php
function call_user_func_array(callable $callback, array $args): mixed
```

`call_user_func_array()` 是 Elephc 支持的杂项内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$callback` (`callable`)
- `$args` (`array`)

**返回值**：`mixed`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
