## ptr()

```php
function ptr(mixed $value): mixed
```

通过具现化可寻址的局部/全局存储的地址来转换 `ptr(value)`。

**参数**：
- `$value` (`mixed`)

**返回值**：`mixed`

_尚无示例 — 请查看 `examples/` 和 `showcases/` 以了解用法模式。_
