## exit()

```php
function exit(int $status): void
```

`exit()` 是 Elephc 支持的 process 内置函数。除非下文另有说明，其行为与 PHP 手册一致。

**参数**：
- `$status` (`int`)，可选

**返回值**：`void`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
