## str_contains()

```php
function str_contains(string $haystack, string $needle): bool
```

Lowers `str_contains()` through `strpos()` and converts found positions to bool.

**Parameters**:
- `$haystack` (`string`)
- `$needle` (`string`)

**Returns**: `bool`
