## str_ireplace()

```php
function str_ireplace(mixed $search, mixed $replace, mixed $subject, int $count): mixed
```

`str_ireplace()` 是 Elephc 支持的 string 内置函数。除非下文另有说明，其行为与 PHP 手册一致。

**参数**：
- `$search` (`mixed`)
- `$replace` (`mixed`)
- `$subject` (`mixed`)
- `$count` (`int`)，通过引用传递，可选

**返回值**：`mixed`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
