## str_replace()

```php
function str_replace(string $search, string $replace, string $subject, int $count): mixed
```

Lowers `str_replace()`/`str_ireplace()` with three string operands.

**Parameters**:
- `$search` (`string`)
- `$replace` (`string`)
- `$subject` (`string`)
- `$count` (`int`), passed by reference, optional

**Returns**: `mixed`
