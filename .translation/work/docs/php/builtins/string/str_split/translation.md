## str_split()

```php
function str_split(string $string, int $length): array
```

将 `str_split(string, length?)` 降级为定宽字符串数组拆分器。

**参数**：
- `$string` (`string`)
- `$length` (`int`)，可选

**返回值**：`array`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
