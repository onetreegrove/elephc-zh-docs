## strcasecmp()

```php
function strcasecmp(string $string1, string $string2): int
```

`strcasecmp()` 是 Elephc 支持的 string 内置函数。除非下文另有说明，其行为与 PHP 手册一致。

**参数**：
- `$string1` (`string`)
- `$string2` (`string`)

**返回值**：`int`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
