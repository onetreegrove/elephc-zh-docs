## strpos()

```php
function strpos(string $haystack, string $needle, int $offset): mixed
```

Lowers `strpos()`/`strrpos()` and boxes position-or-false results as Mixed.

**Parameters**:
- `$haystack` (`string`)
- `$needle` (`string`)
- `$offset` (`int`), optional

**Returns**: `mixed`
