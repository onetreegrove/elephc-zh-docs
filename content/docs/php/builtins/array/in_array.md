## in_array()

```php
function in_array(mixed $needle, array $haystack, bool $strict): mixed
```

降级包含标量或字符串负载的索引数组的 `in_array()`。

**参数**：
- `$needle` (`mixed`)
- `$haystack` (`array`)
- `$strict` (`bool`)，可选

**返回值**：`mixed`
