## class_exists()

```php
function class_exists(string $class, bool $autoload): bool
```

检查指定的类是否已定义。

**参数**：
- `$class` (`string`)
- `$autoload` (`bool`)，可选

**返回值**：`bool`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
