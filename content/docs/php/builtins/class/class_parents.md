## class_parents()

```php
function class_parents(mixed $object_or_class, bool $autoload): mixed
```

`class_parents()` 是 Elephc 支持的类内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$object_or_class` (`mixed`)
- `$autoload` (`bool`)，可选

**返回值**：`mixed`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
