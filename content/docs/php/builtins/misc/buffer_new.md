## buffer_new()

```php
function buffer_new(int $length): mixed
```

`buffer_new()` 是 Elephc 支持的杂项内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$length` (`int`)

**返回值**：`mixed`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
