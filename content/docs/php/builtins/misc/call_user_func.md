## call_user_func()

```php
function call_user_func(callable $callback, ...$args): mixed
```

`call_user_func()` 是 Elephc 支持的杂项内置函数。除非另有说明，其行为与 PHP 手册一致。

**参数**：
- `$callback` (`callable`)
- `...$args` — 可变参数：收集多余的参数到 `$args` 中。

**返回值**：`mixed`

_暂无示例 — 请查看 `examples/` 和 `showcases/` 获取使用模式。_
