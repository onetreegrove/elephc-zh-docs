## phpversion()

```php
function phpversion(string $extension = null): string
```

返回当前 PHP 版本信息。

**参数**：
- `$extension` (`string`)，默认值为 `null`，可选

**返回值**：`string`

_暂无示例——请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
