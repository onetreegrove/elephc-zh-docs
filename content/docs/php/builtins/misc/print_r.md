## print_r()

```php
function print_r(...$values): void
```

打印关于变量的易读信息。

**参数**：
- `...$values` — 可变参数：收集多余的参数到 `$values` 中。

**返回值**：`void`

_暂无示例 — 请检查 `examples/` 和 `showcases/` 以了解使用模式。_
