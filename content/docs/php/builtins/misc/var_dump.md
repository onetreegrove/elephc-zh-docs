## var_dump()

```php
function var_dump(...$values): void
```

打印变量的相关信息，包括其类型与值。

**参数**:
- `...$values` — 可变参数：将额外的参数收集到 `$values` 中。

**返回值**: `void`

_暂无示例 —— 请查看 `examples/` 和 `showcases/` 了解使用模式。_
