## rtrim()

```php
function rtrim(string $string, string $characters): string
```

`rtrim()` 是 Elephc 支持的 string 内置函数。除非下文另有说明，其行为与 PHP 手册一致。

**参数**：
- `$string` (`string`)
- `$characters` (`string`)，可选

**返回值**：`string`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
