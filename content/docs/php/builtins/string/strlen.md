## strlen()

```php
function strlen(string $string): int
```

通过将类似字符串的值强制转换为字符串并返回字节长度来降级处理 `strlen()`。

**参数**：
- `$string` (`string`)

**返回值**：`int`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
