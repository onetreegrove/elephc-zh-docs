## is_bool()

```php
function is_bool(mixed $value): bool
```

`is_bool()` 是 Elephc 支持的 type 内置函数。除非下文另有说明，其行为与 PHP 手册一致。

**参数**：
- `$value` (`mixed`)

**返回值**：`bool`

_暂无示例 — 请参阅 `examples/` 和 `showcases/` 目录以了解使用模式。_
